# Base Demonstration Prompt

```text
Design a highly available microservices e-commerce platform.

Requirements:
1. Include an API Gateway for routing.
2. A Product Service backed by PostgreSQL.
3. An Order Service that communicates asynchronously using Kafka.
4. An Inventory Service using Redis for caching.
5. All services are deployed on Kubernetes.

Please generate the software requirements, architecture, and the necessary UML diagrams.
```
